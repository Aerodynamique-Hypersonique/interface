Comment générer un profil

Pour générer le profil, vous pouvez vous appuyez sur les différents modèles existants sur la page, ou importer une plus complexe. Pour cela, vous devez charger un fichier de type tabulaire. C'est-à-dire, un fichier de ce format:

Colonne1 \t Colonne2 \t **X** \t **Y** \t ColonneN
Valeur1 \t Valeur2 \t ValeurX \t ValeurY \t ValeurN

Une fois le fichier généré, vous verrez le profil sur le graphique juste en dessous. A partir de là, les données auront donc été chargé en mémoire, et vous pouvez passer à la partie suivante.
